🎯 关键技术亮点

1. 智能情感检测
   • 千问 ASR 一次性提取文字+情感
   • 无需额外调用情感分析 API
   • 支持 7 种情感类型

2. 多模态融合算法
   • 三通道加权融合（可调节权重）
   • EMA 时间平滑（避免情感跳变）
   • Valence-Arousal 二维情感空间

3. 无障碍 UI 设计
   • 呼吸动画（视觉吸引力强）
   • 红色警报（高对比度，易识别）
   • 金色摘要（突出显示，易阅读）

4. 性能优化
   • 双阶段 Yield（秒级响应）
   • 异步视频生成（不阻塞交互）
   • 前端先出字出声，后台静默生成视频

🎓 使用场景示例

场景 1：职场会议同传
  听障员工参加会议时，使用语音聊天功能实时捕捉同事发言的文字和
  情感，系统自动生成核心意图摘要和高情商应对建议。

场景 2：一对一面谈
  听障员工与领导面谈时，使用视频聊天功能，系统通过面部表情、语
  音语气、文字内容三通道融合分析对方情绪，当检测到愤怒或不满时
  自动触发红色警报。

场景 3：日常沟通
  听障员工日常工作沟通中，完全依赖视觉交互（无音频输出），通过
  金色摘要框快速理解对方意图，提高沟通效率。
